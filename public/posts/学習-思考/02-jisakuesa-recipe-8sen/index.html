市販エサより強い・安い・簡単な自作エサの時代へ  

「エサ代が高い」「いつも同じエサで飽きられてる」

そんな悩みを解決するのが、プロが実際に使用している自作エサです。

配合を工夫すれば：

  • ✅ 市販エサより釣果が出る
  • ✅ コストは1/3以下
  • ✅ その日の状況に応じてカスタマイズできる

この記事では、現場で検証済みの自作エサレシピ8選を、配合比・作り方・使用シーン別に完全公開します。

自作エサレシピ集  

【レシピ1】最強・青物エサ「ハイブリッド沈下餌」  

配合比

  • 冷凍アキアミ（解凍後）: 150g
  • 冷凍オキアミ粉末: 50g
  • 小麦粉: 30g
  • 少量の塩
  • 水: 大さじ1～2

作り方

  1. 冷凍アキアミを解凍し、ボウルに入れる
  2. オキアミ粉末と小麦粉を混ぜる
  3. 塩小さじ1を加えてよく混ぜる
  4. 水を加えながら、硬さを調整
  5. ラップに包んで、冷蔵保管（当日中に使用）

使用シーン

  • 青物狙い、放流直後、活性が高い時間帯
  • 効果: 沈下速度が速く、広がりが少ないため、青物層に直撃

プロのコツ

塩を多めに入れると、水分が抜けて針持ちが良くなる。昼間に活性が落ちた時間帯に特に効果的。

【レシピ2】真鯛必釣「カスタム練り餌」  

配合比

  • 生オキアミ: 100g
  • 冷凍ホタテ（みじん切り）: 30g
  • パン粉: 20g
  • 砂糖（ひとつまみ）
  • 卵白: 小さじ1

作り方

  1. 生オキアミをボウルに入れ、フォークでつぶす
  2. ホタテのみじん切りを加える
  3. パン粉を加え、よく混ぜる
  4. 砂糖と卵白を加え、さらに混ぜる
  5. 冷蔵保管（最大2日）

使用シーン

  • 真鯛狙い、活性が中程度の時間帯
  • 効果: 甘い香りが真鯛を呼び、粘り強さで針持ちも優秀

現場レポ

  • 午前10時～午後1時の活性低下時間に特に有効
  • 他のエサで反応がない場面での最後の一手

【レシピ3】シマアジ激釣「匂い重視ブレンド」  

配合比

  • 本虫（ブラックホール）: 80g
  • 冷凍アキアミ: 100g
  • イカスミ粉末: 小さじ1
  • にんにく粉末: 小さじ1/2

作り方

  1. 本虫と冷凍アキアミを解凍・混ぜる
  2. イカスミとにんにくを混ぜる
  3. ラップに包み、冷蔵庫で30分放置（匂い浸透）
  4. 使用前に軽く揉んで、再度なじませる

使用シーン

  • シマアジ狙い、活性が低い午後
  • 効果: 独特の匂いでシマアジが敏感に反応

上級テクニック

シマアジは匂いに非常に敏感。このエサを使う際は、3投ごとに新しいエサに付け替えると、匂いの拡散効果がアップします。

【レシピ4】根魚特化「硬い沈下餌」  

配合比

  • 冷凍アキアミ: 100g
  • パン粉: 50g
  • 片栗粉: 20g
  • 塩: 小さじ1
  • 水: 大さじ1/2

作り方

  1. アキアミを解凍し、ボウルに入れる
  2. パン粉と片栗粉を別の容器で混ぜる
  3. アキアミに加え、よく混ぜる
  4. 塩と水を加える
  5. 硬くなるまで混ぜ、ラップに包む

使用シーン

  • 根魚狙い、深場探索
  • 効果: 硬さで針持ちが良く、底付けで根魚エリアに長時間ステイ

使い方のコツ

  • ボール状に丸めて、沈下速度を調整
  • ポイント探索時は親指大（標準）、反応が良い場所では小指大にして何度も投げ込む

【レシピ5】コスト最強「スーパー万能餌」  

配合比

  • 冷凍イカ（市販）: 100g
  • スーパーのエビ塩辛漬け: 50g
  • 小麦粉: 30g
  • オリーブオイル: 小さじ1

作り方

  1. イカをみじん切り、ボウルに入れる
  2. エビ塩辛漬けを加える（汁ごと）
  3. 小麦粉を加えて混ぜる
  4. オリーブオイルを加え、馴染ませる
  5. 使用直前に調整

使用シーン

  • 初心者向け、複数の魚種を狙う
  • 効果: どの魚にもそこそこ効く。コスパ最強

予算比較

  • 市販エサ: 1回3,000円
  • 自作（このレシピ）: 1回800円

【レシピ6】放流直後最強「即効性エサ」  

配合比

  • 冷凍アキアミ: 200g
  • 冷凍オキアミ（粉末）: 100g
  • 鶏がらスープの素: 小さじ2
  • 水: 大さじ2

作り方

  1. アキアミを解凍する
  2. オキアミ粉末とスープの素を混ぜ、別容器で用意
  3. 水を加えて香りを立たせる
  4. アキアミに混ぜる
  5. 使用直前に再度混ぜる

使用シーン

  • 朝一限定：放流直後1時間
  • 効果: 集魚力が爆発的。初期段階で大物を確保できる

現場での活用法

朝一に3～4投でも釣れれば、このレシピが効いた証拠。昼間に活性が落ちたら別レシピに切り替え。

【レシピ7】昼間活性低下「秘密兵器」  

配合比

  • 生オキアミ: 150g
  • 本虫: 50g
  • わさび粉末: 小さじ1/4
  • レモン汁: 小さじ1

作り方

  1. 生オキアミを軽くほぐす
  2. 本虫を細かく刻む
  3. わさび粉末を加える
  4. レモン汁を加えて香りを立たせる
  5. 軽く混ぜ、冷蔵保管

使用シーン

  • 午前10時～午後3時の活性低下時
  • 効果: ワサビとレモンの刺激で、活性を一時的に引き出せる

注意点

  • 効果は一時的なため、30分ごとに新しいエサに取り替える
  • 使いすぎると魚が警戒してしまうため、慎重に

【レシピ8】夕方の鮮度落ち対策「保存強化版」  

配合比

  • 冷凍アキアミ: 150g
  • 冷凍ホタテ: 30g
  • 白味噌: 大さじ1
  • 塩: 小さじ1/2

作り方

  1. アキアミとホタテを軽く解凍
  2. 白味噌を加え、よく混ぜる
  3. 塩を加える
  4. ラップに包み、冷蔵保管（最大3日）
  5. 使用直前に温度調整

使用シーン

  • 夕方の釣行、複数回の釣行予定
  • 効果: 味噌が鮮度を保ち、3日保つ

長期保存のコツ

  • 冷蔵庫の一番奥（温度が低い場所）に保管
  • 使用前に常温で5分置いて、温度を馴染ませる

自作エサ作成時の注意点  

衛生管理  

  • 手指をアルコール消毒してから触る
  • 調理器具は毎回洗浄・乾燥させる
  • 保冷剤は必須

鮮度管理  

  • 作成は釣行の前夜が目安
  • 当日中に使い切る（長期保管は上記レシピ8のみ例外）
  • 異臭がしたら絶対に使用しない

効果測定  

  • 初回は「親指大」から始める
  • 反応がなければ、少し小さくして試す
  • 釣れたエサのサイズ・配合を記録する

最後に：自作エサで差がつく  

市販エサが「万能」なのに対し、自作エサは「その日の状況に合わせたカスタマイズ」ができます。

  • 朝一は「即効性」重視
  • 昼間は「匂い」重視
  • 根魚狙いは「硬さ」重視

このレシピ8つをマスターすれば、どんな活性でも対応できる釣り人になります。
