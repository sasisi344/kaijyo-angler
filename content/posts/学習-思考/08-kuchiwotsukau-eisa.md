## シマアジが「口を使わない」理由と、「口を使うエサ」の秘密

「シマアジを狙ってるのに、つつくだけで食わない…」

多くの釣り人が経験する、このストレス。

実は、その理由は **「エサ選びが間違っている」** からです。

シマアジは：
- ✅ つつき癖が強い（複数回つつく）
- ✅ 食べるか食べないかを「慎重に判断」する
- ✅ 鮮度と匂いに、超敏感

この記事では、**シマアジが確実に「口を使う」エサだけを12種類厳選**しました。

## シマアジが「口を使う」エサの条件

| 条件 | 理由 |
|------|------|
| 新鮮度 | 鮮度が悪いと、つつくだけで食べない |
| 大きさ | 小さすぎると飲み込めない、大きすぎると警戒 |
| 匂い | 強すぎず弱すぎず、「自然な海の匂い」が重要 |
| 色 | 白～薄ピンク色が反応しやすい |
| 食感 | 硬すぎず柔らかすぎず、1秒でかみ砕ける |

**これら全てを満たすエサが「シマアジが口を使うエサ」です**

## シマアジ必釣エサ12選

### 【第1位】冷凍スイカ（活きえび）

**特徴**
- シマアジの最愛エサ
- 新鮮度が高く、鮮度管理で釣果が3倍変わる

**選び方**
- 色が鮮やかな赤～ピンク色
- 目が澄んでいる
- 体に傷がない

**付け方**
- 頭に針を丁寧に通す
- ゆっくり底付けで探る
- 活きが良い場合は、触り続ける

**使用時間帯**
- 昼間（11時～16時）
- シマアジが活発な時間帯で最高の効果

**実績**
> 昼12時～14時に、スイカだけで6匹（平均0.9kg）確保

**プロのコツ**
> 「スイカが活きている間は、できるだけ動かさない。自然な動きが重要です」

### 【第2位】生オキアミ（大粒）

**特徴**
- スイカに次ぐ、シマアジの好物
- 活性が少し低い場面でも効く

**選び方**
- 色が薄く透き通っているもの（新鮮）
- 触った時に弾力がある
- 異臭がしない

**付け方**
- 1粒ずつ、丁寧に針に通す
- ゆっくり底から中層へ誘う
- つつき後は、即座に新しいエサに交換

**使用時間帯**
- 午前10時～14時
- シマアジの中程度活性時

**ローテーション戦略**
- 3投ごとに新しいエサに交換
- 1投で複数回つつかれることを前提に

**実績**
> 午前11時～12時、生オキアミで、シマアジ3匹、真鯛1匹

### 【第3位】冷凍ホタテ（小玉）

**特徴**
- 甘い香りでシマアジが好反応
- 白い色が水中で目立つ

**選び方**
- 白くきれいなホタテを選ぶ
- 小ぶりなサイズ（小指大が目安）
- 鮮度が良いもの

**付け方**
- 小さくカットして、針に付ける
- 浮き釣りで層を絞る
- ゆっくり誘う

**使用時間帯**
- 昼12時～15時
- スイカの次の選択肢

**効果**
- スイカの食い渋り対策
- 「エサを変えたら、急に食った」という場面で活躍

**実績**
> スイカで2匹釣った後、ホタテに切り替え → さらに3匹追加

### 【第4位】本虫（ブラックホール）

**特徴**
- 独特の匂いで、活性が落ちたシマアジでも反応
- つつき癖の強いシマアジの「最後の一手」

**選び方**
- できるだけ活き活きしたもの
- 色が濃いものを選ぶ

**付け方**
- 小さく分ける（半分サイズ）
- ゆっくり上下に誘う
- 3投ごとに交換

**使用時間帯**
- 昼13時～16時
- 活性が低下した午後に特に有効

**効果的な場面**
- シマアジ活性が低い
- 他のエサで反応がない
- 時間帯が遅くなっている

**実績**
> 昼14時の活性低下時、本虫で2匹。他エサでは反応なし。

### 【第5位】冷凍むきえび（サイズ中）

**特徴**
- スイカより小さく、食べやすい
- ホタテより新鮮度が保ちやすい

**選び方**
- 色が鮮やかなピンク色
- サイズが均一なもの
- 新鮮度がある（購入日が新しいもの）

**付け方**
- そのまま、または2つに割いて
- 頻繁な交換（2投ごと）
- ゆっくり探り

**使用時間帯**
- 昼11時～15時
- シマアジ狙いの標準的なエサ

**ローテーション例**
- 1投目：スイカ
- 2投目：冷凍むきえび
- 3投目：ホタテ
- 4投目：スイカ（ループ）

**実績**
> 昼間4時間で、むきえびだけで4匹確保

### 【第6位】冷凍小えび（活き状）

**特徴**
- 冷凍でも、解凍後は「活き」に近い動き
- 自然な動きでシマアジが敏感に反応

**選び方**
- 解凍時に色が保たれるもの
- サイズが小ぶり（標準）
- パッケージに「活き状」と書かれているもの

**付け方**
- そのまま、または頭に針を通す
- 中層～底で、ゆっくり誘う
- 動きが止まったら交換

**使用時間帯**
- 昼10時～16時
- 1日中対応できる

**コスパ評価**
- ⭐⭐⭐⭐⭐
- 安い割に、釣果が出る

### 【第7位】冷凍スイカ（脚）

**特徴**
- スイカの脚（触手部分）
- 本体より柔軟性があり、シマアジが食べやすい

**選び方**
- 脚が分離しているもの
- 色が鮮やかな赤色
- 傷がないもの

**付け方**
- 脚を1本、針に通す
- ゆっくり探る
- つつき後、即座に新しいエサ

**使用シーン**
- 本体スイカで反応がない場面
- 「エサを変えたら食った」の事例多数

**実績**
> スイカ本体で2投反応なし → 脚に変更 → 1投で食った

### 【第8位】冷凍オキアミ（粉末配合）

**特徴**
- 粉末を配合エサに混ぜる
- 集魚効果と、食べやすさの両立

**使い方**
- 冷凍アキアミと混ぜる
- 小麦粉を加えて固さ調整
- 団子状にして、底に投げ込む

**配合比（参考）**
- 冷凍アキアミ：80g
- オキアミ粉末：20g
- 小麦粉：10g

**使用シーン**
- 複数の魚種を狙う場面
- シマアジ＋真鯛の両立狙い

**実績**
> 配合エサで、シマアジ3匹、真鯛2匹（同時間帯）

### 【第9位】冷凍刺身用イカ（小切り）

**特徴**
- 小さくカット可能
- シマアジのサイズに合わせられる

**選び方**
- 白～薄ピンク色のもの
- 鮮度が良いもの
- 臭みがないもの

**付け方**
- 小さく（爪の大きさ程度）にカット
- 頻繁な交換（2投ごと）
- ゆっくり探り

**使用シーン**
- スイカが手に入らない場面
- 代替エサとして活躍

**実績**
> 小切りイカで、昼間3時間で3匹

### 【第10位】冷凍マキエビ

**特徴**
- 小ぶりで、シマアジが食べやすい
- 針持ちが良く、エコノミー

**選び方**
- サイズがS～M（小ぶり）
- 色が鮮やかなピンク
- 新鮮度がある

**付け方**
- 頭に針を通す
- ゆっくり底付け
- 5投ごとに交換

**使用シーン**
- スイカ類が手に入らない
- コスパ重視

**実績**
> 終日マキエビで、シマアジ4匹確保

### 【第11位】生タコ（刺身用）

**特徴**
- 独特の食感で、シマアジが敏感に反応
- 新鮮度が高く、鮮度管理で効果が倍変わる

**選び方**
- 白～薄紫色のもの
- 触ってプリッとしたもの
- 新鮮度がある

**付け方**
- 小さくカット
- ゆっくり誘う
- 3投ごとに交換

**使用シーン**
- シマアジが食い渋る場面
- 「最後の一手」として活躍

**コスト**
- 高め（1投で30～50円相当）

**実績**
> 他のエサで反応なし → タコに変更 → 2投で1匹

### 【第12位】冷凍スイカ（ペア）

**特徴**
- 2尾同時付け
- ボリュームで視覚的アクションをプラス

**使い方**
- スイカ1尾を2匹の針に付ける
- または、異なる針に2尾付ける
- ゆっくり誘う

**使用シーン**
- 活性が高い朝方
- ゴールデンタイムの1時間限定

**実績**
> ペアスイカで、朝8時から30分で3匹

## シマアジの「つつき癖」を克服する戦略

### つつくだけで食わない理由

| 理由 | 対策 |
|------|------|
| エサが大きい | 小さくカット |
| エサが古い（鮮度低下） | 新しいエサに交換 |
| 針が大きい | 小さい針（1.5号）に変更 |
| 誘い方が速い | ゆっくり探る |
| エサの匂いが強すぎる | 匂い弱めのエサに切り替え |

### つつき後の対応方法

**パターンA：つついて食わない場合**
1. 1投目のつつきで確認 → 落とす
2. 2投目で新しいエサを投入
3. 3投目で逆サイドに投げ込む
4. 必ず新しいエサに交換

**パターンB：つついて食った場合**
1. 竿は動かさない
2. 2回目のつつきを待つ
3. 2回目のつついた瞬間に、軽く合わせ

## 活性別・時間帯別のエサ使い分け

### 高活性時（朝6時～9時）
**推奨順**
1. スイカ（ペア）← メインで攻める
2. 冷凍むきえび
3. ホタテ

### 中活性時（午前10時～12時）
**推奨順**
1. スイカ（1尾）← 標準的に使用
2. 生オキアミ
3. 冷凍小えび

### 低活性時（昼13時～15時）
**推奨順**
1. 本虫 ← 活性低下対策
2. タコ
3. ホタテ

### 活性回復時（夕方16時～閉場）
**推奨順**
1. スイカ ← 活性が戻る
2. 冷凍むきえび
3. 生オキアミ
