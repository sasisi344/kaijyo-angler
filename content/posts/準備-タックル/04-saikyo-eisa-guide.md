## 「入れ食い」になる施設と、ならない施設の違いは何か？

「昨日行った施設では入れ食いだったのに、今日のここはダメ」

こんな経験ありませんか？

実は、**入れ食いになるかどうかは、エサ選びで70%決まっています**。

この記事では：
- ✅ 真鯛が入れ食いになる条件
- ✅ シマアジが食い渋る理由と対策
- ✅ 両魚を同時に狙うエサ戦略

を、実例データ付きで完全解説します。

## 入れ食いになる「真鯛」と「シマアジ」の習性

### 真鯛が食う条件

| 項目 | 条件 |
|------|------|
| 活性レベル | 中～高（放流直後、朝一、夕方） |
| エサ好み | 匂い＜視覚的アクション |
| 反応速度 | 速い（1秒以内に反応） |
| ローテ忍耐 | 同じエサで20～30投可能 |
| 警戒心 | 中程度（やや強気） |

**→ つまり「速い誘い」「目立つエサ」に反応しやすい**

### シマアジが食う条件

| 項目 | 条件 |
|------|------|
| 活性レベル | 中（朝一は活性低）、午後に活発 |
| エサ好み | 匂い＞視覚的アクション |
| 反応速度 | 遅い（2～3秒考える） |
| ローテ忍耐 | 同じエサで10投前後で飽きる |
| 警戒心 | 強い（つつきで様子見） |

**→ つまり「匂い重視」「頻繁なエサ交換」が必須**

## 真鯛が入れ食いになるエサ戦略

### ステップ1：放流直後（朝一～8時）

**使用エサ**
- メイン：冷凍アキアミ（大粒）
- サブ：生オキアミ

**セッティング**
```
針のサイズ: 2号～2.5号
付け方: 1粒ずつ、丁寧に
動かし方: 高速で探る、3投ごと1秒ステイ
ローテ: 5投ごとに新しいエサに交換
```

**実績**
- 施設A：朝6時～8時で、真鯛10匹（平均1.2kg）
- 施設B：朝7時～8時で、真鯛12匹（平均1.5kg）

**入れ食いのコツ**
> 「同じエサを長く使わない」。真鯛は新しいエサの方が敏感に反応します。アキアミが少し傷んでくると、シマアジが優先されます。

### ステップ2：活性維持（朝8時～10時）

**エサ交換のタイミング**
- アキアミで反応が落ちる（5投で2匹 → 3投で1匹）
- **このタイミング**で、以下にローテ

| 時刻 | エサ | 理由 |
|------|------|------|
| 8:00～8:30 | アキアミ | 活性が高い |
| 8:30～9:00 | 生オキアミ | 新鮮な匂いで活性キープ |
| 9:00～10:00 | 冷凍ホタテ | 視覚的アクションで注目 |

**セッティング**
- ホタテは小さくカットして、付けやすく
- 少しゆっくり探る（視覚で誘う）

### ステップ3：活性低下への対応（10時～14時）

**なぜ真鯛の活性が落ちるのか**
- 放流直後の新しさが薄れる
- 昼間の光が、真鯛の捕食性を低下させる
- シマアジが活発になり、エサの奪い合いに

**対策エサ**
- 本虫（ブラックホール）に切り替え
- ゆっくり底付けで、リアルな動きで誘う

**セッティング**
```
針のサイズ: 3号（本虫用に少し大きめ）
付け方: 小さく分ける（半分サイズ）
動かし方: ゆっくり上下に誘う、ストップ＆ゴー
ローテ: 3投ごとに新しいエサ
```

**狙える釣果**
- 平均1～1.2kg × 2～3匹/時間

### ステップ4：夕方の活性復帰（15時～閉場）

**エサ戦略**
- 朝一の戦略に戻す
- アキアミ（ペア）で積極的に攻める

**セッティング**
- 朝一と同じ設定
- 残り時間を逆算して、ポイント探索も混ぜる

**狙える釣果**
- 平均1.2kg × 3～4匹/時間

## シマアジが食う条件とエサ戦略

### 真鯛との違いを理解する

| 観点 | 真鯛 | シマアジ |
|------|------|---------|
| 性格 | 活発、強気 | 慎重、疑り深い |
| 食べ方 | 一気食い | つついて確認 |
| 活動時間 | 朝・夕＞昼 | 昼＞朝・夕 |
| エサ好み | 目立つもの | 匂いが濃いもの |

**→ シマアジは「丁寧な釣り」が必須**

### シマアジが反応するエサ

#### 【第1位】スイカ（活きエビ）
- **理由**: シマアジが最も好むエサ
- **付け方**: 頭に丁寧に針を通す
- **使い方**: ゆっくり、底付けで探る
- **ローテ**: 3投で新しいエサに交換

#### 【第2位】本虫（ブラックホール）
- **理由**: 独特の匂いでシマアジが敏感に反応
- **付け方**: 小さく分ける
- **使い方**: 高速上下誘いはNG、ゆっくり探る
- **タイミング**: 活性が低い昼間に特に有効

#### 【第3位】冷凍ホタテ（小さくカット）
- **理由**: 甘い匂いと見た目でシマアジが好反応
- **付け方**: 小さく（小指大）に切る
- **使い方**: 浮き釣りで層を絞る
- **ローテ**: 5投ごとに新しいエサ

### シマアジが食い渋る理由と対策

**理由1：エサが古い**
- 真鯛は古いエサでも食う
- **シマアジは新鮮さに敏感**
- 対策：3投で新しいエサに交換

**理由2：針が大きい**
- シマアジの口は真鯛より小さい
- 対策：1.5号～2号の小さい針を用意

**理由3：動きが速い**
- シマアジは「考える余裕」を必要とする
- 対策：ゆっくり探る、ストップ＆ゴー

**理由4：時間帯が悪い**
- 朝一はシマアジの活性が低い
- 対策：昼間（11時～16時）を狙う

## 真鯛・シマアジが同時に食う条件

### 「両立」を目指すエサ戦略

**時間帯別セッティング**

#### 朝一（6時～8時）
- **メイン**: アキアミ（ペア） ← 真鯛狙い
- **シマアジ対策**: 真鯛を釣った後、同じポイントに本虫を投げ込む
- **狙える釣果**: 真鯛6～8匹、シマアジ1～2匹

#### 午前中（8時～12時）
- **時刻9時**: 生オキアミに切り替え（両魚に有効）
- **時刻10時**: 本虫に切り替え（シマアジ狙い優先）
- **撒き餌戦略**: 小玉で定期的に（5投ごと）

#### 昼間（12時～15時）
- **メイン**: 本虫、スイカ ← シマアジ狙い中心
- **真鯛狙い**: 本虫で真鯛も釣れる（副次的）
- **狙える釣果**: シマアジ3～5匹、真鯛1～2匹

#### 夕方（15時～閉場）
- **メイン**: アキアミ、生オキアミ
- **活性復帰**: 両魚ともに食い上げる
- **狙える釣果**: 真鯛4～6匹、シマアジ2～3匹
