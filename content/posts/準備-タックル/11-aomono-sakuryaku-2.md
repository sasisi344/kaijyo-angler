## 釣果が3倍変わる「エサ選びの法則」とは？

海上釣り堀で青物を狙っているのに、なぜか釣果が出ない。

その理由は、実は **「3つの法則」を知らないだけ** です。

この記事では、釣果を3倍に変える：
- ✅ エサ選びの「黄金法則」
- ✅ 仕掛けの「秘密」
- ✅ ローテーションの「パターン」

を、実例データ付きで完全公開します。

## 法則1：「エササイズの法則」が釣果を左右する

### 青物が反応するエササイズ

```
時間帯別・最適エササイズ
├─ 朝6時～8時（朝一ゴールデンタイム）
│  └─ 最適：親指～人差し指サイズ
│     └─ 実例：冷凍イワシM、アキアミペア
│
├─ 朝8時～11時（活性維持タイム）
│  └─ 最適：小指～親指サイズ
│     └─ 実例：冷凍マキエビ、生オキアミ
│
├─ 昼11時～15時（活性低下時間）
│  └─ 最適：小指サイズ＋小ぶり
│     └─ 実例：本虫（小）、マキエビ（小）
│
└─ 夕方15時～閉場（活性回復時間）
   └─ 最適：親指～人差し指サイズに戻す
      └─ 実例：新鮮なアキアミペア
```

### 失敗例vs成功例

**失敗：エササイズを固定化**
```
朝6時～夕方まで、ずっとアキアミペア
↓
朝一：👍（好反応）
昼間：❌❌（反応ゼロ）
夕方：❌（復帰に時間がかかる）

理由：昼間は活性が低いため、
      大きいエサだと「脅威」に見える
```

**成功：時間帯でエササイズを変更**
```
朝一：アキアミペア（大きい）
↓
昼間：本虫＋マキエビ（小さい）
↓
夕方：新しいアキアミペア（大きい）

結果：朝2匹 → 昼1匹 → 夕方2匹
      合計5匹（通常は2～3匹）
```

## 法則2：「エサの匂い濃度の法則」

### 活性に応じた「匂い濃度」の選び方

```
活性レベル ―― 最適な匂い濃度 ―― 推奨エサ
│
高活性（朝一） ―― 中程度 ―― アキアミ、イワシ
    ↓
中活性（午前） ―― 中＋強 ―― 生オキアミ
    ↓
低活性（昼間） ―― 強い ―― 本虫
    ↓
回復活性（夕方） ―― 中程度に戻す ―― アキアミ
```

### 具体的な効果測定

**朝一にアキアミで5投**
```
反応：5投で3匹 = 60%の反応率
```

**同じ時間帯に「匂いが強すぎる本虫」を使用**
```
反応：5投で1匹 = 20%の反応率
理由：活性が高すぎて、匂いが「過剰」に感じられ、
      青物が警戒してしまう
```

**昼間（活性低）に本虫を使用**
```
反応：5投で1匹 = 20%の反応率
     （他のエサでは0匹だったため、相対的に有効）
```

## 法則3：「エサ交換タイミングの法則」が最重要

### 活性別・最適交換タイミング

```
活性が「高い」場合
└─ 1投ごとに新しいエサに交換
   理由：古いエサは警戒される
   実績：交換なし → 3投で1匹
       毎投交換 → 3投で3匹（3倍）

活性が「中」の場合
└─ 3投ごとに交換
   理由：バランスが取れたタイミング

活性が「低」の場合
└─ 5投ごと（またはそれ以上）に交換
   理由：古いエサでも、匂いが強いため有効
```

### 実例による釣果比較

**朝一の同じ1時間を、異なる交換タイミングで実施**

| 交換タイミング | 釣果 | 相対評価 |
|-----------|------|--------|
| 交換なし（同じエサで投げ続ける） | 2匹 | 基準 |
| 5投ごと | 4匹 | **2倍** |
| 3投ごと | 5匹 | **2.5倍** |
| 1投ごと | 6匹 | **3倍** |

**結論：1投ごとに新しいエサに交換で、釣果が3倍になる**

## 法則4：「ローテーションパターンの法則」

### 最強ローテーションパターン

**パターンA：朝一専用（活性が最も高い時間）**

```
1投目：冷凍アキアミペア
  ↓ 反応があれば即座に交換
2投目：新しい冷凍アキアミペア
  ↓ 反応があれば即座に交換
3投目：冷凍イワシ に切り替え
  ↓
1投目へループ
```

**効果**
- 5投で3匹が基準
- パターン確立による「安定釣果」

**パターンB：午前中用（活性が少し落ちる）**

```
1投目：生オキアミ
2投目：冷凍マキエビ
3投目：冷凍アキアミペア（新しい）
  ↓
ループ（ポイント移動しながら）
```

**効果**
- 5投で2匹程度
- 「ローテで活性をキープ」戦略

**パターンC：昼間用（活性が最も低い）**

```
1投目：本虫
2投目：新しい冷凍アキアミペア（試す）
3投目：本虫に戻す
  ↓
ポイント移動後、ループ
```

**効果**
- 5投で1匹が期待値
- 「昼間は難しい」と割り切ったパターン

**パターンD：夕方専用（活性回復）**

```
1投目：新しい冷凍アキアミペア
2投目：冷凍イワシ
3投目：新しい冷凍アキアミペア
  ↓
朝一パターンに戻す
```

**効果**
- 5投で2～3匹
- 朝一より活性は落ちるが、「回復傾向」

## 法則5：「仕掛けの法則」（針サイズ＆付け方）

### 青物専用仕掛けの秘密

**仕掛けの構成**

```

    ↓
```

### 針サイズの重要性

| 針サイズ | 対象 | 釣果 | 評価 |
|--------|------|------|------|
| 2号 | 真鯛・シマアジ | 反応する | 青物には小さすぎ |
| 3号 | 小型青物 | 反応する | 中型は反応落ちる |
| **4号** | **全青物** | **最高** | **バランス型** |
| 5号 | 大型青物 | 最高 | 吸い込みが難しい |

**→ 4号が最適（全青物対応）**

### エサの付け方の工夫

**付け方で釣果が変わる**

**方法1：頭に刺す（標準）**
- 真っすぐ、丁寧に
- 活き具合を最大限保つ
- 釣果：基準値

**方法2：背に刺す＋ペア（ダブル）**
- 2尾を同時に付ける
- ボリュームが出る
- 釣果：**1.3～1.5倍**
- 朝一専用

**方法3：腹に刺す（活き重視）**
- 活きの良さが最大限保たれる
- 一匹の寿命が長くなる
- 釣果：1.1倍（わずかだが有効）
